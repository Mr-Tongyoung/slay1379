## 📊 상권 분석 웹 서비스

LG U+의 통신사 데이터를 활용하여 지역 상권의 유동 인구 및 방문 인구를 분석하고, 상권 유형을 분류 및 추천해주는 데이터 기반 웹 서비스입니다.

## 🔍 주요 기능

- 유동 인구 및 체류 인구 데이터 기반 상권 분석
- 상권별 주요 지표 시각화 (체류 시간, 방문 비율 등)
- K-Means 기반 상권 유형 클러스터링
- 사용자 맞춤형 커스텀 지표 생성 및 저장
- 시계열 데이터 기반 변화 추이 분석
